# Projeto 1 - Conversão de Temperatura e Cálculo de Fatorial

Este projeto foi desenvolvido como parte da disciplina de Programação de Computadores.

## Objetivo

Criar dois programas simples em Python:

1. Conversor de temperatura de Celsius para Fahrenheit.
2. Calculadora de fatorial utilizando a biblioteca nativa `math`.

## Tecnologias utilizadas

- Python
- Biblioteca nativa `math`
- Entrada de dados com `input()`
- Conversão de tipos com `float()` e `int()`
- Saída de dados com `print()`
- Uso de f-string
